# Supabase_Chat

Supabase_Chat is a chat application that allows users to send messages and images seamlessly.

## Features

- **Send Messages**: Easily send text messages to other users.
- **Send Images**: Share images in the chat with just a few clicks.

## Installation

To install and set up the Supabase_Chat application locally, follow these steps:

1. **Visit the Installation Link**: https://drive.google.com/file/d/1TLLnCr_51GLuhFzCNzV-0Hy_gKvJqzVw/view?usp=drive_link
2. **Register**: Sign up for an account using your email.
3. **Enjoy**: Start chatting with others instantly.

## Usage

Once registered, you can log in and start sending messages and images to other users in the chat.

## Technologies Used

- **Frontend**: (e.g., Flutter)
- **Backend**: (e.g., Supabase, Dart)
- **Database**: (e.g., Supabase)
- **Authentication**: (e.g., Supabase Authentication)
